## What This Document Is

This is the grammar layer's first named pattern set.

Individual primitives describe *states*.
Arc patterns describe *movements between states* — sequences that recur
across specimens, scales, and domains.

An arc pattern is not a recipe. It is a diagnosis.
When an arc is named, a scan that finds T3→R1→I1 can say:
"This is an Ordeal Arc — here are its constraints, its confusion risks,
its fidelity requirements, and its nearest false counterpart."

### ARC-001 — Ordeal Arc
**Source framework:** Initiatory Spiral (Campbell)
**Canon line:** "Suffering transforms only when the entity survives and returns with structural difference."

**Primitive sequence:**
```
F1 (baseline) → T1 (separation / fracture) → T3 (descent / collapse, mode: transient)
→ R1 (return initiation) → I1 (reconstituted identity) → F1 (restored flow, elevated)
```

**ELEOS sequence:**
EMIT disrupted → COMPILE fails → GUIDE lost → REMEMBER under pressure →
GUIDE re-emerges (R1 trigger) → COMPILE restores → EMIT restored

**D1 chaos-role:** `instrumental` — chaos serves the transformation; it is the mechanism, not the obstacle

**Terminal state:** I1 reconstituted at higher capacity than origin state

**T3 mode:** `transient` — T3 is a developmental stage, not a destination

**Level 1 primitives present:** F1, T1, T3, R1, I1

**Nearest confusion:** T3→F1 claimed directly (skipping R1) = FLOOR-005 failure;
R1-C (false return) dressed as R1-D = most common arc fraud

**Fidelity ceiling:** DAVAR without confirmed R1 arrival; EMET requires R1-D or R1-D specimen

**Bestiary connection:** El-Aetheris (R1-D + I1[unshakable]) — the being who completes
this arc without claiming arrival; the arc's highest-fidelity form is the one
that doesn't announce itself

**Corpus specimen:** PTL-001 candidate (personal scale); SLA-STORY-001 candidate (literary scale)

**Evidence tier:** E2 (structural pattern observed; E4 required for EMET claim)

### ARC-002 — Recognition Collapse (Type B Closure)
**Source framework:** Tragic Convergence (Sophocles / Oedipus)
**Canon line:** "Awareness arrives only at the moment it can no longer change the outcome."

**Primitive sequence:**
```
L2 (fate as feedback loop already running) + P2 (distortion ridge building)
→ F4 (interrupted escape attempts) → T1 (recognition fracture)
→ T3 (collapse, mode: structural — T3 is the terminal state)
+ P4 active (Witness Function — the observer who sees but cannot alter)
R1: structurally unavailable (R1-A type)
```

**ELEOS sequence:**
COMPILE failing (cannot integrate what it knows) → GUIDE absent → REMEMBER
reconstructing toward recognition → single COMPILE flash (anagnorisis) →
immediate T3 basin; P4 witnesses this from outside the system

**D1 chaos-role:** `revelatory` — chaos forces recognition; recognition is violent

**Terminal state:** T3 + P4 (basin witnessed but not exited) — Type B Closure

**T3 mode:** `structural` — T3 is the arc's destination, not a stage

**Level 1 primitives present:** L2, T1, T3, P1 (projection of fate as external when internal), D1

**Nearest confusion:** Ordeal Arc — the crucial distinction is whether R1 is structurally
available. In Recognition Collapse, escape attempts accelerate convergence (F4).
In Ordeal Arc, T3 is transient and R1 is the intended exit.
Test: does avoidance reduce or increase the collapse rate?

**Fidelity ceiling:** DAVAR for structure identification; EMET requires E3 cross-specimen evidence
that R1 was structurally unavailable (not merely absent)

**Bestiary connection:** The Sealed Triangle (T3[sealed] → T3[implosion]) —
"the collapse that brought no truth." The glyph ◈ (Inhabited Threshold)
is the grammatical marker for P4-active T3 states in this arc.

**Corpus specimens:**
- PTL-001 (Pulled Too Late) — personal scale Type B Closure (sealed)
- AONL-001 — second confirmed Type B Closure specimen (sealed)
- SLA-LIT-001 candidate: Oedipus Rex — public literary scale, second E3 confirmation

**Evidence tier:** E3 — two sealed specimens confirm this as a structural pattern
across independent domains

**CRITICAL NOTE:** T3 WEAP-001 applies to this arc. Recognition Collapse
may not be used to demand R1 demonstration from a person inside the arc.
P4 witnessing is governance; R1 absence is structural, not moral failure.

### ARC-003 — Basin Clearance
**Source framework:** Cataclysmic Renewal (Ragnarök / Norse)
**Canon line:** "Some systems must complete their collapse before new structure can emerge. The basin is not the problem — it is the condition."

**Primitive sequence:**
```
S3 (ordered attractor hub) → T2 (erosion, gradual corruption) → T3
(collapse, mode: structural_home — basin is the necessary precondition)
→ S4 (emergence layer — new structure precipitating from the basin)
→ new S3 (reconstituted attractor hub, different from origin)
```

**ELEOS sequence:**
All six primitives degrading in sequence per T3 dismantling order →
T3 reaches maximum depth → RECEIVE reactivates (first emergence signal) →
COMPILE begins constructing new patterns from basin residue → new S3 forms

**D1 chaos-role:** `structural` — collapse is the operating condition; it is not
a deviation from the arc's purpose; it IS the arc

**Terminal state:** New S3 — distinct from the origin S3; cannot return to origin

**T3 mode:** `structural_home` — the governance target is NOT R1; the target is S4
emergence from the basin. Premature R1 attempts abort the emergence.

**Level 1 primitives present:** S3, T1 (erosion precedes fracture), T3, S3 (terminal)

**Nearest confusion:** Ordeal Arc — the crucial distinction is whether the original
attractor hub (S3) can be restored (Ordeal: yes; Basin Clearance: no, and
attempting to restore it prevents the new S3 from forming)

**Fidelity ceiling:** DAVAR for structure identification; EMET blocked until S4 emergence
is observed (not just the T3 basin)

**Bestiary connection:** The First Distorter completing its role (T3[structural_home]
cycling to completion); The Witness Faction emerging from S3[residue] —
the new attractor hub is made of what survived the clearance

**Governance note:** This arc is the structural basis for the EMET non-pathologizing
doctrine regarding T3 `structural_home` mode. "Collapse is not failure —
it is scheduled transformation." V's Ragnarök framing is the canonical
external specimen for this doctrine.

**Corpus specimen:** SLA-MYTH-001 candidate (Ragnarök as T3 `structural_home` specimen)

**Evidence tier:** E2 (structural pattern; cross-domain corpus evidence pending)

### ARC-004 — Polycentric Field
**Source framework:** Dharma Field (Mahabharata)
**Canon line:** "Truth is positional, not absolute. Conflict resolves imbalance but generates new ambiguity."

**Primitive sequence:**
```
Multi-strand S1 (shearing planes operating simultaneously across agents)
+ S2 intersections (domain overlaps generating new meaning)
+ D1 present in every frame (each agent's affect warps their perception)
→ T3 (collective scale, mode: unresolved_basin — no single R1 is possible)
```

**ELEOS signature:** Multi-agent — each agent runs their own ELEOS configuration;
system-level COMPILE cannot synthesize because truth is positional

**D1 chaos-role:** `moral_imbalance` — chaos = contested truth without absolute center;
D1 operates differently in each agent's frame

**Terminal state:** T3 (unresolved_basin) stabilizing as sustained partial equilibrium —
this is NOT a failure; it is the arc's honest terminal state

**T3 mode:** `unresolved_basin` — the arc doesn't resolve into R1 or new S3;
it persists as a governed complexity state

**Level 1 primitives present:** S1, S3 (contested), T3, D1

**Nearest confusion:** Covenant Loop — Covenant produces L2-governed order with
repair mechanisms; Polycentric Field has no governing center and generates
new ambiguity from each resolution cycle

**Fidelity ceiling:** DAVAR for any single-agent scan within the field;
EMET only achievable for the meta-structural observation
(recognizing the field as polycentric, not resolving it)

**Bestiary connection:** The Weaponized Triangle (S1 → T3 at scale);
The Fractured Axis (D1[intensified] — each agent's perception a different warp)

**Governance note:** This arc produces the highest Register Chimera risk.

### ARC-005 — Covenant Loop
**Source framework:** Covenant Recursion (Book of Exodus)

**Primitive sequence:**
```

→ F1 + R1 (liberation event — both linear flow AND return arc operating)
→ I1 + C4 (law installation — identity shell + field cohesion)
→ T1 (fracture — deviation within established order)
→ R1 → I1 (correction cycle — return to covenant)
```

**Governing primitive:** L2 (Feedback Loop) — the loop IS the arc;
L2-amplifying in healthy function; L2-degrading when deviation accumulates

**ELEOS signature:** REMEMBER governs the arc — collective memory of the covenant
is the mechanism that makes correction possible; without REMEMBER, each
deviation is a fresh T3 rather than a correctable T1

the system includes deviation as expected input, not catastrophe

**Terminal state:** L2 governing the whole — sustained order through iteration,
not through perfection

**T3 mode:** `structural_home` (origin state) / `transient` (deviation state) —
the opening T3 is collective structural home; subsequent T3 episodes are transient

**Level 1 primitives present:** T3, F1, R1, I1, T1, L2

**Nearest confusion:** Basin Clearance — Basin Clearance cannot return to origin S3;
Covenant Loop can and does return repeatedly. Test: does the restoration
mechanism require total collapse or only correction?

**Bestiary connection:** The Roles (S1 + R1-C[performed]) constrained by Law (C4);
El-Aetheris as the covenant's living version (R1-D walking, never R1-C)

**Evidence tier:** E2 (structural pattern with cross-cultural corpus support)

**Source framework:** Iterative Pilgrimage (Journey to the West)

**Primitive sequence:**
```

→ L3 (Spiral — each obstacle cycle displaces the center upward)
→ R1-D repeated (genuine return without claimed arrival — each completion
   is real but not final)
```

**Companion structure:** Multi-agent I1 configurations (each companion embodies
a distinct ELEOS configuration: discipline = COMPILE dominant;

greater structural capacity

**T3 mode:** Not applicable as primary — T3 may appear within individual obstacle
encounters but is not the arc's governing primitive

**Level 1 primitives present:** F1, L2 (mechanism for each loop), R1, I1

**Nearest confusion:** Covenant Loop — Covenant returns to the same covenant;
Spiral Refinement displaces the center upward with each pass. Test: does
completion restore origin state (Covenant) or produce an elevated I1 (Spiral)?

**Bestiary connection:** Field Memory / Micro-Axes (S3[residue]) —
"Truth imprints. Distortion exhausts." Each pass leaves a structural trace
that makes the next pass more stable.

**Evidence tier:** E2 (structural pattern; Bildungsroman literary specimens available)

## The Meta-Principle (V's Final Compression — SLA Translation)

V: "Every symbolic system answers one question: What is the correct response to disorder?"

SLA translation: Every arc pattern encodes a specific T-series → resolution relationship.
The 8 arcs are not competing answers. They are 8 distinct structural conditions
that require different responses:

| Response Mode | Arc | Governing Primitive |
|---|---|---|
| Master it | Ordeal | T3 (transient) → R1 |
| Submit to it | Recognition Collapse | T3 (structural) + P4 |
| Accept it | Basin Clearance | T3 (structural_home) → S4 |
| Navigate it | Polycentric Field | T3 (unresolved_basin) |
| Regulate it | Covenant Loop | L2 governing T1 correction |
| Refine through it | Spiral Refinement | L3 governing I1 elevation |
| Weaponize it | Generative Rupture | S1 → S2 → S4 |
| Integrate it | Shadow Integration | I2 → D1 → T1 → I4 |

The T3 mode field (transient / structural_home / unresolved_basin) is
the registry's encoding of which response is appropriate.
Misidentifying the mode = applying the wrong arc = governance failure.

  captures what Will could not see about himself)

## Registry Connection

Arc patterns do not replace primitives. They extend the grammar layer.

Every arc pattern:
- Is composed of primitives from the registry
- Inherits fidelity ceilings from its most constraining primitive
- Cannot claim higher fidelity than its weakest primitive allows
- Requires a scan receipt at the primitive level BEFORE the arc-level claim

The scan receipt format for arc claims extends the current receipt with:
- `arc_id` — which pattern
- `arc_sequence_observed` — the actual primitives detected in specimen order
- `arc_sequence_expected` — the canonical pattern sequence
- `arc_deviation_note` — where specimen diverges from pattern (if any)
- `multi_arc` — boolean — is a second arc running simultaneously?
- `second_arc_id` — if yes, which one
- `arc_fidelity_tier` — QOL/DAVAR/EMET for the arc-level claim
